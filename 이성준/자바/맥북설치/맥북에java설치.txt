<2> 다운로드 링크
https://www.oracle.com/java/technologies/downloads/
에서 스크롤을 아래로 내리면 예전 버전이 나온다.

여기서 Java11버전을 선택하고 macOS를 선택한다.

jdk-11.0.16.1_macos-x64_bin.dmg 를 다운로드 받는다.

Document도 아래 경로서 다운로드 받는다.
https://www.oracle.com/java/technologies/javase-jdk11-doc-downloads.html

<3>설치파일은 Finder 의 다운로드 폴더에서 확인하여
설치파일을 더블클릭하면 설치화면이 나옴

<4> Java 환경변수 설정
- 터미널앱을 실행시킨다.
터미널에서
ls -al 입력하여 파일 목록 확인
cd /Library/Java/JavaVirtualMachines 입력하여 해당경로로 들어가서 설치된 jdk를 확인하고,
해당 경로에서 cd (설치된jdk이름).jdk/Contents/Home 입력 후,
해당 경로를 확인하는 명령어 pwd 를 입력하여 경로를 확인합니다.

- 복사한 경로를 .bash_profile 에 붙여넣어 JAVA 경로를 잡아줍니다.
vi ~/.bash_profile 입력하여 vi 편집기를 통해 .bash_profile을 열어 아래 내용을 기술한다

export JAVA_HOME=/Library/Java/JavaVirtualMachines/(설치된jdk이름).jdk/Contents/Home

다 입력 후 esc 클릭하여 보기모드로 나온 후,​
:(shift+;) 를 입력하여 나가기 모드를 합니다.
나가기 모드에서 wq! 를 입력하여 저장 후 나가면 됩니다.
(w:저장하기, q:나가기, wq:저장 후 나가기) 
- 환경변수 설정이 끝나면 java -version 을 입력하여 java 버전을 확인합니다.

2. 맥북에서 이클립스 설치
참조사이트: https://eunoia3jy.tistory.com/25
[1] 다운로드
https://www.eclipse.org/downloads/packages/
위 사이트에서 2번째 목록인
Eclipse IDE for Enterprise Java and Web Developers 의 macOS x86_64 를 클릭합니다.
(Eclipse IDE for Java Developers 로 설치하면 안된다. 반드시 Enterprise 용으로 설치하자)

[2] Finder 에서 eclipse-jee-~macosx- xxxx  .dmg 설치파일을 확인
dmg파일을 더블클릭> eclipse아이콘을 Applications폴더안에
끌어다 넣는다.=> 설치됨
런치패드에 eclipse아이콘이 생긴다. 이를 클릭한다
>열기버튼 클릭
>workspace경로를 선택> Launch버튼 클릭

터미널
$ sdk version  <— sdkman 버전출력  
 $ sdk list java  <— 설치 가능 & 설치된 JDK목록  
 $ sdk install java 11.0.12.7.2-amzn <— 지정된 JDK설치(원하는 종류와 버전 지정)  
 $ sdk default java 11.0.12.7.2-amzn <— 사용할 java버전을 변경(모든 쉘에 적용)
 $ sdk use java 11.0.12.7.2-amzn <— 사용할 java버전을 변경(현재 쉘에만 적용)  
 $ sdk current java <— 현재 사용중인 java버전 출력  
 $ echo $JAVA_HOME  <— JAVA_HOME으로 지정된 경로 출력
